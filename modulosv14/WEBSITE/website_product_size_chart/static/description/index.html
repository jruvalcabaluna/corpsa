WEBSITE: Product Size Chart for every Product
Providing Easy To Use Size Chart Guide For Customers !

Features:

  • Provide the dynamic size chart guide .
  • Display size options .
  • Display measurement information .
  • Display size chart related image.
  • Help in Convening the Customer .

  • Website View

  • Setup & Configure

Website View

User can view the size chart link on product page.
In side the popup available size options for product can be displayed.
Measurement related information will available in next tab.

Setup & Configure

Under Website ==> Configuration ==> Size Chart menu admin can create size chart.
Here admin can assign size chart to multiple products at once.
Admin can also add Size chart guide and measurement guide here.
